# Product Description

## What It Does

Good First Issue is a Flutter mobile app that helps newcomers to open source discover beginner-friendly GitHub issues. The app aggregates issues labeled "good first issue" from various open source projects, making it easier for new contributors to find their first contribution opportunity.

## Who Uses It

- **New open source contributors** looking for beginner-friendly issues to work on
- **Developers learning to contribute** to open source projects
- **Anyone wanting to give back** to the open source community but unsure where to start

## Key Features

- **GitHub OAuth Authentication** — Secure sign-in via Supabase GitHub OAuth
- **Issue Discovery** — Browse and search "good first issue" labeled issues by organization
- **Organization-based Search** — Filter issues by specific GitHub organizations
- **Issue Details** — View full issue descriptions, labels, and metadata
- **Direct Links** — Quick access to issues on GitHub for immediate contribution

## Business Context

This is an open source project aimed at lowering the barrier to entry for open source contribution. By curating beginner-friendly issues, it helps grow the open source community and makes projects more accessible to new developers.
